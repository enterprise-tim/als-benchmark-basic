üß™ Test Scenarios

The benchmark suite includes these test scenarios:

  • Small Object (1KB): Basic AsyncLocalStorage usage with minimal data
  • Medium Object (10KB): Moderate data size to test scaling
  • Large Object (100KB): Large data objects to test memory impact
  • Nested Contexts: AsyncLocalStorage.run() calls within other ALS contexts
  • Concurrent Operations: Multiple simultaneous ALS operations
  • Memory Leak Detection: Long-running tests to detect memory growth
